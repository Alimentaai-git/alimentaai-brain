# Identidade — Stack

## Ferramentas de identidade

| Artefacto | Detalhe |
|-----------|---------|
| Tipografia | **Inter** (`fontFamily.sans` no Tailwind do site) |
| Cores | Tokens **HSL** em `:root` no ficheiro `src/index.css` do site (ex.: `--primary`, `--secondary`, `--muted`, `--accent`, `--destructive`, `--foreground`, `--background`) |
| Ícones | Font Awesome (sólidos + marcas, ex. WhatsApp) |
| Componentes UI | shadcn/ui sobre Radix |

## Referências visuais

- Gradientes de marca usados na hero e CTAs: `from-primary to-secondary` e variantes (ver landing no repo do site).
- Modo escuro: variáveis `.dark` no mesmo `index.css`.
